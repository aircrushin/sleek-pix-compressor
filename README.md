# Sleek Pix Compressor

![License](https://img.shields.io/badge/license-MIT-blue.svg)

一个现代化的图片压缩工具，支持多语言和暗黑模式，提供直观的压缩质量调节功能。

## 功能特性

- 🖼️ 支持拖放和文件选择上传图片
- ⚡ 实时压缩预览
- 🎚️ 可调节压缩质量 (0-100%)
- 🌓 自动跟随系统暗黑模式
- 🌐 支持中英双语切换
- 📊 显示原始大小、压缩后大小和压缩比例
- 📥 一键下载压缩后的图片

## 技术栈

- ⚛️ React
- 🎨 Tailwind CSS
- 🛠️ TypeScript
- 🌐 i18next 多语言支持
- 🚀 Vite 构建工具

## 快速开始

### 安装依赖

```bash
npm install
```

### 运行

```bash
npm run dev
```
